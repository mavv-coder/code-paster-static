Code Paster

Code paster is a free online app

What can I use it for?

You can use Code Paster to share code or any kind of text with partners, students, colleagues, etc. The trainer (who could be one or more people) has the right to write and read the text but the student only have access to read the text typed by the trainer. So that, it is a great tool to use in online classes.

How does it work?

  1. Press the button "Get started" to navigate to the app
  2. Presh "Create session" to a private room for you and your people
  3. Press the button "Get started"
